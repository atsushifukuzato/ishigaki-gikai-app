# 石垣－基隆航路とやいま丸

## 目的

このフォルダは、Topic「石垣－基隆航路とやいま丸」に関する資料を、
一般質問、行政執行、報道、その他資料に分けて整理し、
後から高品質な `topic.json` を作成できる状態にするための作業基盤です。

この Topic は、単独議案追跡型ではなく、一般質問・行政執行・報道ベースで
成立するテーマとして扱います。議案は、明確に関連が確認できた場合のみ
補助的に扱います。

## このフォルダの使い方

1. `docs/sources/ishigaki-keelung-route-yaimamaru/` 配下に資料を保存します。
2. PDF や URL を追加したら、まず `source-notes.md` に要点をメモします。
3. 日付が分かる資料は `timeline.md` に時系列で追記します。
4. 論点、未確認事項、出典の精度がそろってから Topic JSON を作成します。

## ファイルの役割

- `source-notes.md`
  収集した資料の要点、論点、出典、未確認事項をまとめる作業メモです。
- `timeline.md`
  いつ何が起きたかを月単位で追記し、`topic_updates` の下書きに使う時系列メモです。

## 資料保存先

- `docs/sources/ishigaki-keelung-route-yaimamaru/council/`
  一般質問 PDF、議会資料、議事録メモ
- `docs/sources/ishigaki-keelung-route-yaimamaru/city/`
  市公式 PDF、行政資料、事業説明資料
- `docs/sources/ishigaki-keelung-route-yaimamaru/news/`
  新聞 PDF、記事保存資料
- `docs/sources/ishigaki-keelung-route-yaimamaru/other/`
  上記以外の参考資料

## 運用メモ

- 出典 URL は必ず残します。
- 断定できない内容は、事実と推測を分けてメモします。
- bill との紐付けは、正式名称と一次資料が確認できた場合のみ行います。
