# 石垣－基隆航路とやいま丸 source notes

## 1. Topic概要

- 概要:
  石垣市と台湾・基隆を結ぶ定期貨客船航路計画、および就航船「やいま丸」をめぐる行政執行、港湾・CIQ整備、損失補償、就航延期、有事利用の論点を整理する Topic。
- 対象期間:
  2023年末ごろから 2026年4月時点までを中心に確認する。
- 主な主体:
  石垣市、株式会社商船やいま、石垣市議会、沖縄県、国、港湾・CIQ 関係機関、地元金融機関。
- 基本出典:
  市議会一般質問資料、補正予算・議案資料、市公式資料、港湾・CIQ 関連資料、八重山毎日新聞、YaimaLine 公式サイト。
  現時点では、石垣市資料 `台湾基隆港との定期航路開設に向けて（令和７年２月）` が行政側の基礎資料として重要。

## 2. 主な論点

- 公費負担・損失補償:
  石垣市が商船やいまの借入に対してどの範囲で信用補完するのか。
- 就航準備と延期:
  修繕、回航、検査、内装変更、手続きの遅れが就航時期にどう影響しているか。
- 港湾・CIQ整備:
  暫定CIQ施設、クルーズ岸壁利用、貨物船岸壁利用、第2バース要望との関係。
- 採算性・経済性:
  市の見込み、運航便数、寄港・物流・観光への影響をどう説明しているか。
- 有事利用:
  やいま丸が国民保護訓練上の候補船として扱われることの位置づけ。
- 議会での論点:
  一般質問や予算審議で、何が「確認された事実」で何が「議員の問い」か。
- 事業体制:
  検討委員会、準備課、新法人（株式会社商船やいま）、船舶購入契約の流れ。

### 通告書ベースで存在が確認できる問い

- 運航開始日はいつか。
- 運賃はいくらか。
- 就航が遅れている理由は何か。
- 基隆港での停泊料金はいくらか。
- 商船やいまの融資条件、金利、返済期間、返済方法はどうなっているか。
- 使用岸壁、使用施設、CIQ・荷役業者との調整はどう進んでいるか。
- やいま丸と有事・大規模災害対応の関係をどう位置づけるか。
- 第三セクターとしての位置づけや、市と商船やいまの関係をどう説明するか。

### まだ断定しないほうがよい事項

- 2025年9月就航予定が正式決定だったのか、それとも計画段階の見込みだったのか。
- 2026年2月以降のクルーズ岸壁利用が最終確定なのか、当時点の計画なのか。
- 融資や停泊料金の詳細条件。
- 運賃の具体額。
- 事業採算の実現可能性評価。
- 有事利用が制度上どこまで正式に位置づけられているか。現時点で確認できるのは「規制緩和があれば可能性がある」という答弁まで。

### 9.1 `description` 下書き

  石垣市と台湾・基隆を結ぶ定期航路計画と就航船「やいま丸」をめぐり、石垣市は検討委員会設置、準備課新設、候補船選定、新法人支援、港湾・CIQ調整、損失補償を含む支援を進めてきた。一方で、就航時期、採算性、港湾利用、公費負担、有事・住民避難との関係などが議会や報道で継続的に論点化している。

  2026-02-07T00:00:00+09:00
  理由: 現在確認済みの公開資料のうち、もっとも遅い日付の主要資料が 2026-02-07 の報道であるため。将来、2026-03 の会議録が確認できたら更新候補。

- 第1段落:
  航路計画の概要。石垣市と基隆港を結ぶ定期貨客船航路、やいま丸、商船やいま、検討委員会から準備課設置まで。

  議会論点。有事利用、住民避難、第三セクター性、支援の妥当性。
- 書かない方がよいこと:
  運賃の具体額、最終就航日、停泊料金、制度上確定していない有事利用の断定。
