## いまの整理

- 今回の実務上の主対象は、`bills` / `bill_contents` に載せる「議案」と「議員提出議案」
- `document_type` 上は `report` / `consent` / `approval` も扱えるが、請願は現行スキーマに直接の種別がない
- そのため、まずは次の範囲を優先して整備するのが自然
  - 市長提出の議案第5号〜第35号
  - 議員提出議案第1号〜第12号
