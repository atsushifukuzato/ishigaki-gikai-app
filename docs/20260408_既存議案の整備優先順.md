# 既存議案の整備優先順

## 方針

既存議案は、まず以下の順で整備する。

1. トップに出ている注目議案
2. `published` の議案
3. `coming_soon` の議案
4. `draft` の議案

公開中の議案を先に整えることで、利用者が実際に見るページから順に品質を上げる。

## 次に整える優先5件

### 1. 学校給食無償化促進議案
- 市民への影響が最も伝わりやすい
- 子育て・教育分野の代表例として使いやすい
- すでに `normal` / `hard` の内容があるため、推奨構成への整理がしやすい

### 2. 船荷証券の電子化に関する法律案
- 産業・物流の観点でテーマがはっきりしている
- 「何が変わるのか」を整理すると読みやすくなる余地が大きい
- 関連リンクも載せやすい

### 3. ガソリン税暫定税率廃止議案
- 生活コストに直結し、関心を持たれやすい
- メリット・デメリットが混在しているため、「主な論点」と相性がよい
- normal / hard の差分も作りやすい

### 4. こども家庭庁予算大幅増額議案
- 予算議案として重要
- 「必要な理由」「影響を受ける人」を追加すると理解しやすくなる
- 財源や実施効果の整理で深さを出しやすい

### 5. 中学生・高校生向けプログラミング教育必修化議案
- 否決案件の例として、結果の理由を説明しやすい
- 「主な論点」が特に重要な議案
- 可決案件だけでなく、否決案件の見せ方も確認できる

## 各議案で入れる推奨構成

`bill_contents.content` では、以下の見出し構成を基本とする。

1. この議案のポイント
2. この議案が必要な理由
3. 主な論点
4. 影響を受ける人
5. 関連リンク

## 作業手順

1. 管理画面で議案コンテンツ編集を開く
2. `テンプレートを挿入` を押す
3. 既存の `normal` 内容を見出し構成に整理する
4. 必要に応じて `hard` も同様に整理する
5. 保存後、公開ページで表示確認する

## 補足

- `normal` は市民向けの標準版として、短文・平易な言い換えを優先する
- `hard` は制度や財政、法的背景まで含めた詳細版とする
- `関連リンク` は、公式資料や会議録など信頼できる一次情報を優先する
